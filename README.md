# Advance-Crop-Disease-Detection-using-Deep-Leaning
Advanced Crop Health Monitoring using Machine Learning

An AI-powered crop disease detection system that leverages Machine Learning and Deep Learning techniques to identify plant diseases from leaf images. The project uses Convolutional Neural Networks (CNNs) for image classification, enabling early disease detection and improving agricultural productivity. It includes image preprocessing, model training, disease prediction, and a user-friendly Streamlit web application for real-time crop health monitoring. The system helps farmers and agricultural experts make faster, data-driven decisions, reducing crop loss and supporting smart farming practices. Based on scalable AI architecture, the project can be extended with IoT integration, cloud deployment, mobile applications, and support for multiple crop species and diseases.

Tech Stack: Python, TensorFlow, Keras, OpenCV, NumPy, Pandas, Streamlit, CNN, Deep Learning, Machine Learning.
